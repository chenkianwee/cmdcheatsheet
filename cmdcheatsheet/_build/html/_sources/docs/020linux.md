# Linux commands

## Ubuntu
Update the OS
```
$ sudo apt-get update

$ sudo apt-get upgrade
```
Remove unnecessary files after an OS upgrade.
```
$ sudo apt-get autoremove
```
Install an app
```
$ sudo apt-get app_name
```
List a directory
```
$ ls
```
List with details, or show all hidden files
```
$ ls -l file/foldername

$ ls -la file/foldername
```
Remove a file.
```
$ rm filename
```
Remove a folder.
```
$ rm -r foldername
```
